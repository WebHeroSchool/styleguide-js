# styleguide-js

### 1. Выравнивание

Для выравнивания блоков кода используются табуляции (1 табуляция на 1 уровень выравнивания), знаки табуляции не используются.

Плохо:
```css
(function() {
var name = 'Skywalker';
return name;
})()
```
Хорошо:
```css
(function() {
  var name = 'Skywalker';
  return name;
})();
```
